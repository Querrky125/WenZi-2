# 🚀 朵朵工具箱

一个现代化的实用工具集合网站，包含4款精心设计的实用工具。

## ✨ 特色功能

- 🎨 **动态背景**：使用Vanta.js实现飞鸟动画效果
- 🌈 **霓虹渐变**：标题文字带有动态渐变和故障特效
- 💫 **交互设计**：工具卡片悬停时有发光和抬升效果
- 📱 **响应式布局**：完美适配各种设备屏幕

## 🎯 工具列表

包含4款实用工具：

### 📱 文本转二维码
- 快速将文本、网址等内容转换为二维码
- 支持下载保存
- 高清晰度输出

### 🎲 随机数生成器
- 支持自定义数值范围
- 可生成多个随机数
- 支持不重复模式
- 保存历史记录

### 📏 单位换算工具
- 支持长度、重量、温度、面积、体积换算
- 常用单位全覆盖
- 实时换算结果
- 高精度计算

### 📝 单词听写助手
- 多难度等级：小学、初中、高中、大学四级
- 真人语音朗读
- 自动批改和统计
- 错题列表复习功能

## 🚀 部署到Vercel

1. Fork本项目到你的GitHub
2. 在Vercel中导入项目
3. 点击部署即可

或使用Vercel CLI：
```bash
npm i -g vercel
vercel
```

## 💻 本地开发

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 🛠️ 技术栈

- 纯HTML/CSS/JavaScript
- Tailwind CSS
- Three.js
- Vanta.js
